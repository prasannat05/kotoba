# Kotoba – Learn Japanese

## Overview
Kotoba is a structured Japanese learning application designed to help users master Hiragana and Katakana through an interactive, flashcard-based system combined with quizzes, writing practice, and progress tracking.

- App Link : https://tinyurl.com/Kotoba-app
## Core Features

### Alphabet Mastery (Hiragana and Katakana)
- Complete coverage of 46 base characters along with Dakuten and Handakuten.
- Flashcard-based learning system presenting one character at a time.
- Displays large Japanese character, Romaji, and phonetic guide for clarity.
- Sequential navigation with swipe gestures and manual controls.
- Option to mark characters as learned for tracking mastery.

### Dynamic Learning Structure
- Content organized into logical chapters such as Vowels, K-row, S-row, etc.
- Chapters are dynamically generated based on the dataset.
- Scalable design allows easy addition of new content without UI changes.

### Vocabulary and Sentences
- Categorized vocabulary including Greetings, Numbers, Colors, Family, Food, and Places.
- Collection of commonly used daily sentences for practical usage.
- Context-aware learning flow integrated with quizzes.

### Quiz and Assessment System
- Multiple-choice questions for:
  - Character to Romaji recognition
  - Romaji to character identification
  - Vocabulary and sentence translation
- Immediate feedback for each answer.
- Chapter-wise quizzes with final score summary.
- Accuracy tracking for performance evaluation.

### Writing Practice Module
- Canvas-based drawing interface for practicing characters.
- Faint background reference to guide stroke formation.
- Controls to clear and retry drawings.
- Supports iterative learning through repetition.

### Progress Tracking and Analytics
- Tracks completed chapters and learned characters.
- Records quiz scores and accuracy percentages.
- XP-based progression system rewarding user activity.
- Daily streak tracking to encourage consistency.
- Displays progress through visual indicators and statistics.

### Dashboard and User Flow
- Central dashboard showing:
  - Total XP
  - Current streak
  - Resume learning option
- Smooth navigation from learning to assessment and progress review.
- Clear and structured learning journey from beginner to intermediate level.

### UI and User Experience
- Built using Material 3 design principles.
- Clean and minimal interface focused on distraction-free learning.
- Japanese-inspired color palette with balanced typography.
- Smooth transitions and responsive layouts across devices.
- Supports both light and dark themes.

### Offline Capability
- Fully functional without internet connectivity.
- All data stored locally for fast and reliable performance.

## Technical Highlights

- MVVM architecture for separation of concerns and scalability.
- Jetpack Compose for modern declarative UI.
- Room database for local data persistence.
- StateFlow for efficient state management.
- Jetpack Navigation for structured screen transitions.
- Optimized for Android devices running API 26 and above.

## Limitations

- Pronunciation is text-based with no audio support.
- Writing practice does not include advanced recognition or AI-based evaluation.
- Data is stored locally with no cloud synchronization.
